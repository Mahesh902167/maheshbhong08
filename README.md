# Ecommerce Sales Dashboard (Power BI)

## 📊 Project Overview
This dashboard shows sales performance of an ecommerce business.

## 🔑 Key Insights
- Total Sales: 438K
- Total Profit: 37K
- Top Category: Clothing
- Highest Sales Month: January

## 📌 Features
- Sales by Category
- Profit by Month
- Payment Mode Analysis
- Sub-category performance

## 🛠 Tools Used
- Power BI
- Excel

## ⚠️ Challenges Faced

- Faced issues with data cleaning and handling missing values in the dataset  
- Difficulty in understanding dataset structure and relationships between tables  
- Errors while creating DAX measures like Profit and Average Order Value (AOV)  
- Confusion in selecting appropriate visuals (bar chart, pie chart, etc.)  
- Initial dashboard design was not clean and properly aligned  
- Performance issues due to large data size  
- Slicer and filter synchronization was not working properly  

## ✅ Solutions

- Cleaned and transformed data using Power BI Query Editor  
- Built proper relationships between tables for better analysis  
- Practiced DAX formulas and created accurate measures  
- Selected visuals based on business requirements  
- Improved dashboard UI using themes and proper alignment  
- Removed unnecessary columns to optimize performance  
- Fixed filter and slicer settings for proper interaction
